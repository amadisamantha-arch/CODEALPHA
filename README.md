## Projects Completed

### Task 1 — Iris Flower Classification
- Trained a K-Nearest Neighbors (KNN) classification model to identify 
  three species of Iris flowers based on sepal and petal measurements
- Achieved 93.33% accuracy (96.67% at optimal K value)
- 🔗 [View Repository](https://github.com/amadisamantha-arch/CodeAlpha_IrisFlowerClassification)

### Task 2 — Unemployment Analysis with Python
- Analysed unemployment trends across Indian states from 2019–2020
- Investigated the impact of Covid-19 on unemployment rates
- Key finding: Unemployment peaked at 23.24% in May 2020 post-lockdown
- 🔗 [View Repository](https://github.com/amadisamantha-arch/CodeAlpha_UnemploymentAnalysis)

### Task 3 — Car Price Prediction with Machine Learning
- Built and compared three regression models (Linear Regression, 
  Random Forest, Gradient Boosting) to predict used car selling prices
- Random Forest achieved R² score of 96.22% with MAE of ₹0.62 Lakhs
- 🔗 [View Repository](https://github.com/amadisamantha-arch/CodeAlpha_CarPricePrediction)
